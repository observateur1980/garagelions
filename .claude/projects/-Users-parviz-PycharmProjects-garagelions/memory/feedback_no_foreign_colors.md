Never introduce colors from garageliving.com or other reference sites into the garagelions codebase. Only use the existing garagelions brand palette.

**Why:** User was frustrated that dark blue (#0b2545) from garageliving.com was added to their CSS. Reference screenshots are for layout/structure inspiration only, not color copying.

**How to apply:** When replicating a design from a reference site, map colors to garagelions' own palette: primary orange #c64d1e, charcoal gray #626262, light grays #eeeeee/#f9f9f9, white #ffffff. Never copy hex values from the reference.
